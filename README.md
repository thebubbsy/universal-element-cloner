# Universal Element Cloner

A high-fidelity Chrome/Edge browser extension for cloning DOM elements with advanced features including auto-scroll, style freezing, and an interactive canvas editor.

## Features

- **Element Picker**: Select single or multiple elements with visual feedback
- **Full Page Capture**: Automatically capture entire pages with lazy-loaded content
- **Auto-Scroll**: Intelligent scrolling to capture dynamically loaded elements
- **Style Freezing**: Preserve computed styles for pixel-perfect exports
- **Interactive Canvas**: Edit, move, resize, and delete captured elements
- **Theme Toggle**: Dark/Light mode support
- **Export**: Save captures as standalone HTML files

## Tech Stack

- **JavaScript ES6+** - Modern async/await patterns
- **Chrome Extension Manifest V3** - Latest extension platform
- **Vanilla CSS/HTML** - No external dependencies

## Installation

### From Source
1. Clone this repository
2. Open Chrome/Edge and navigate to `chrome://extensions/` or `edge://extensions/`
3. Enable "Developer mode"
4. Click "Load unpacked"
5. Select the extension directory

## Usage

1. Click the extension icon to open the side panel
2. Choose your capture mode:
   - **Element Picker (Canvas)**: Select elements to edit on canvas
   - **Export Element (Direct)**: Select and export immediately
   - **Full Page Capture**: Capture the entire page
3. Use the canvas editor to refine your capture
4. Export as HTML
